# 美素佳儿DP-AI复盘系统

## 项目简介

美素佳儿DP-AI复盘系统是一个基于网页的应用程序，用于分析销售数据与主播排班的关系。系统可以自动匹配订单与主播的工作时段，生成销售分析报告，并提供数据可视化和AI驱动的销售建议。该系统专为抖音直播电商场景设计，通过整合直播销售数据、主播排班表及复盘文档，利用AI分析主播话术质量、互动节奏与排班效率，自动生成针对性优化方案与定制化话术指南，驱动主播能力提升与销售转化率增长，实现"数据归因-精准诊断-策略迭代"闭环管理。

## 主要功能

1. **文件上传与预处理**
   - 支持拖放上传Excel文件
   - 支持销售数据、主播排班表和主播月度资料的上传
   - AI文件预处理功能，可以按照指令自动清洗和整理数据
   - 实时显示文件上传和处理状态

2. **数据分析**
   - 自动匹配订单与主播工作时段
   - 生成销售统计和分析报告
   - 计算关键业务指标
   - 按商品类别（源悦、莼悦、旺玥、皇家）分析销售情况

3. **可视化展示**
   - 24小时销售趋势图
   - 主播销售业绩对比图
   - 商品类别分析图表
   - 交互式数据表格
   - 支持按时间段筛选销售数据

4. **AI智能分析**
   - 整体销售趋势分析
   - 个性化主播销售建议
   - 商品类别销售分析
   - AI聊天助手，可以回答关于数据分析的问题

5. **结果导出**
   - 支持导出分析结果为Excel文件
   - 支持导出单个主播的订单数据
   - 支持导出AI分析报告
   - 包含详细的匹配数据和统计信息

## 使用说明

### 1. 数据格式要求

#### 销售数据Excel格式
必须包含以下字段：
- 主订单编号
- 订单提交时间（格式：YYYY-MM-DD HH:MM:SS）
- 选购商品
- 订单应付金额

#### 主播排班表Excel格式
表格结构要求：
- 第一行：日期列标题（例如：3月1号、3月2号等）
- 第一列：主播列（主播姓名/ID）
- 单元格内容：时间段格式（例如：8:00-9:00）
- 无排班时间段使用"-"表示

示例：
```
主播      | 3月1号     | 3月2号     | ...
好运      | 8:00-9:00 | 9:00-10:00 | ...
诗诗      | 13:00-14:00| -         | ...
素心      | 17:00-18:00| 18:00-19:00| ...
```

#### 主播月度资料Excel格式（可选）
应包含以下信息：
- 主播姓名/ID
- 粉丝数量
- 历史转化率
- 月度业绩指标
- 其他绩效数据

### 2. 操作步骤

1. 打开系统网页
2. (可选) 使用AI文件预处理功能处理源数据
   - 上传待处理的Excel文件
   - 输入处理指令（例如：保留特定字段、删除空值行等）
   - 点击"开始处理"
   - 下载处理结果或直接用于分析
3. 上传销售数据Excel文件
4. 上传主播排班表Excel文件
5. (可选) 上传主播月度资料Excel文件
6. 点击"开始分析"按钮
7. 等待分析完成
8. 查看分析结果和图表
9. 使用AI聊天助手获取更深入的分析
10. (可选) 导出分析结果

### 3. 查看结果

分析结果分为多个部分：
- **匹配结果**：显示订单与主播的匹配详情，支持分页查看
- **数据分析**：展示销售趋势、主播业绩对比和商品类别分析
- **AI智能点评**：提供销售分析和改进建议
- **商品类别分析**：按产品系列（源悦、莼悦、旺玥、皇家）展示销售情况
- **销售时段分析**：24小时销售趋势分析

### 4. AI聊天助手使用

1. 完成数据分析后，AI聊天助手会自动激活
2. 点击屏幕角落的AI助手图标打开聊天窗口
3. 输入问题，如"哪个主播的销售最好？"或"哪个时段销售额最高？"
4. AI助手会根据分析结果回答问题
5. 可以继续追问或提出新的问题

## 技术架构

### 前端架构
- **核心技术**：原生JavaScript
- **UI框架**：Bootstrap 5
- **图表库**：Chart.js
- **Excel处理**：SheetJS (XLSX)
- **样式**：CSS3
- **AI交互**：基于API调用或模拟数据

### 功能模块
- **文件上传模块**：处理文件拖放和读取
- **数据处理模块**：解析Excel文件和数据匹配
- **可视化模块**：生成图表和数据展示
- **AI分析模块**：提供智能分析和建议
- **导出模块**：生成Excel报告

### 数据流程
1. 用户上传数据文件
2. 系统解析文件内容
3. 执行数据匹配算法
4. 生成分析结果和统计数据
5. 渲染图表和报表
6. (可选) 调用AI服务获取建议
7. 用户可导出结果或通过AI助手交互

## 常见问题

1. **文件上传失败**
   - 检查文件格式是否为.xlsx或.xls
   - 确认文件大小是否在限制范围内
   - 验证文件是否损坏

2. **数据匹配结果不准确**
   - 检查时间格式是否正确
   - 确认主播排班时间是否完整
   - 验证销售数据的时间戳格式
   - 尝试在设置中调整时间容差值

3. **图表显示异常**
   - 清除浏览器缓存
   - 刷新页面
   - 使用其他现代浏览器尝试

4. **AI功能无法使用**
   - 确认是否已完成数据分析
   - 检查API配置是否正确
   - 可切换到演示模式使用基本功能
